# vscode-theme-ocean
🎨 an ocean style theme of vscode

![](icon.png)

Vscode Market: https://marketplace.visualstudio.com/items?itemName=alanlang.theme-ocean#review-details

## 安装
1. 打开`vscode`插件管理
2. 搜索 `Ocean Theme`
3. 下载安装
